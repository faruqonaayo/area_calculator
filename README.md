# Area Calculator

A simple Python program to calculate the area of various geometric shapes using command-line arguments.

## Features

- Calculate area for:
    - Rectangle
    - Square

- Simple command-line usage

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/area_calculator.git
cd area_calculator
```

## Usage

Run the program with command-line arguments:

```bash
python main.py --height = <height> -- width = <width>
```

Replace `<height>`, `<width>`, with your desired values.

## Example

```bash
python main.py --height 5 --width 3
# Output: Area of shape: 15.0

```
